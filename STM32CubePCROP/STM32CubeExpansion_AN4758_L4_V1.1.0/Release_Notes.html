Purpose

The goal of this X-CUBE-PCROP firmware is to illustrate the practical usage of the PCROP protection feature.

For more details on PCROP implementation in microcontrollers, refer to the corresponding application note :

AN4758 Proprietary code read-out protection on STM32L4, STM32L4+, STM32G4, and STM32WB series MCUs
